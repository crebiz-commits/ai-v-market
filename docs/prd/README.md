## 작성 기준
- **as-built**: 추측 없이 실제 소스/마이그레이션을 읽고 `file:line` 근거로 기술.
- **검증 우선**(CLAUDE.md): 코드에서 확인 못 한 항목은 "※ 검증 필요"로 표기.
- 작성 중 발견된 실제 불일치/부채는 각 문서 "알려진 제약/이월" 절에 기록.
