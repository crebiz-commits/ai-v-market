# 🎬 크리에이터 유치 계획 (Creator Acquisition Plan)

> **목표:** CREAITE에 실제 AI 영상 크리에이터들이 영상을 올리게 만든다 (공급 측 콜드스타트 해결).

## 0. 문제 정의 — 두 시장 콜드스타트

CREAITE는 양면 플랫폼: **시청자**는 볼 게 많아야 오고, **크리에이터**는 볼 사람(+돈)이 있어야 올린다.
- 현재 데모/쇼케이스 ~100편으로 "비어 보이지 않게"는 해둠(시청자용 미끼).
- **진짜 병목 = 실제 크리에이터의 첫 업로드.** 이 문서는 그걸 푼다.

**불편한 진실(정직하게 전제):** 신규 플랫폼은 초기 트래픽이 적어 **광고·구독 수익이 초반엔 미미**하다. 그리고 **실결제·현금 정산은 토스 가맹 심사(1~2개월) 후**에야 가능. → 그러니 초기 후크를 "당장 큰돈"이 아니라 **① 라이선스 판매라는 새 수익원 ② 초기 입점 특별대우 ③ AI 영상 전용 무대(전시·포트폴리오)** 로 잡는다.

## 1. 왜 CREAITE에 올리는가 (포지셔닝 — 확정)

핵심 카피(전단 확정): **"플랫폼은 가져가고, CREAITE는 돌려드립니다. — 당신을 한 분의 배급사로."**

| 구분 | 유튜브/넷플릭스 등 거대 플랫폼 | CREAITE |
|---|---|---|
| 수익 배분 | 플랫폼이 대부분 가져감 | 라이선스 **80%** / 광고 **50~60%** / 구독료풀 **50%** 환원 |
| **라이선스 판매** | ❌ 없음 | ✅ **AI 영상을 상업용 자산으로 판매**(유튜브·SNS·기업 마케팅 사용권) — *최대 차별점* |
| AI 영상 전용 무대 | 일반 영상에 묻힘 | AI 시네마 전문 큐레이션·전시 |
| 진입 | — | 앱 설치 없이 웹에서 즉시 업로드 |

> **요지:** 광고·구독은 트래픽 따라오는 후행 수익. **초기 크리에이터에게 파는 건 "라이선스 판매 + 작품 무대 + 초기 특혜"**다.

## 1.5 자동화 vs 수동 — ⚠️ 무엇을 자동화하고 무엇은 사람이 하나 (검증됨 2026-06-18)

**원칙: 자동화 우선. 단, 콜드 아웃리치(낯선 사람에게 먼저 연락)는 자동화 금지 — 한국에선 불법 + 계정 정지.**

- **정보통신망법 제50조**: 영리목적 광고성 정보는 **수신자 명시적 사전 동의 필수**. **2026-03-12 통과 개정안에서 제재 강화 — 위반 시 매출액 6% 이하 과징금.** 동의 없는 대량 DM/문자/메일 = 불법 스팸.
- **플랫폼 약관(인스타·유튜브)**: 동의 없는 자동 DM = 스팸 → 차단·섀도밴·**계정 영구삭제**(2026 봇 차단 강화). 합법 자동 DM은 "최근 24h 내 상호작용자"만 — **콜드 자동 DM 불가.**

| 영역 | 방식 | 비고 |
|---|---|---|
| 유료 광고(Google·Meta·카카오 타겟팅) | ✅ **완전 자동·합법·무한확장** | "AI 영상 제작자" 타겟 → 랜딩 → 가입. **1순위 자동 엔진. 예산 월 30~50만원(2026-06-18 확정).** |
| 레퍼럴 루프(제품 내장) | ✅ 자동 | 크리에이터→크리에이터 초대 보상 |
| 무마찰 온보딩 | ✅ 자동 | 도착→가입→업로드→AI홍보문건→공유 |
| SEO·공개 콘텐츠 마케팅 | ✅ 자동/반자동 | "AI 영상 수익화" 유입. *광고 아닌* 유익한 공개 글은 스팸 아님 |
| 아웃리치(후보 리스트·AI 초안·CRM) | 🟡 **세미 자동** | 도구는 자동, **발송은 공개게시/진성 소량 1:1/동의기반만** |
| 콜드 대량 DM·문자·메일 | ❌ **금지** | 불법(과징금)+계정밴. 절대 하지 않음 |

**전략 전환:** 스케일 엔진 = **유료광고+레퍼럴+무마찰 온보딩(자동)**. 아래 Phase 1 수동 50인은 *스케일 수단이 아니라* **첫 코호트 품질·커뮤니티 호감용 소량 보조**로 격하.

## 2. 단계별 실행 계획

### Phase 0 — 업로드 마찰 제거 (출시 전, 우리가 직접 코드 / 1~2주)
*근거: 업로드 흐름 코드 점검(Upload.tsx). 목표 = 첫 업로드 전환율 ↑*

- [ ] **로그인 장벽 완화** — 현재 업로드 진입에 로그인 강제(`Upload.tsx:459`). 최소한 **업로드 폼·가치(수익 80%)를 비로그인도 끝까지 보게** 하고, 제출 직전 소셜 원클릭 로그인으로. (완전 익명 업로드는 스토리지/보안 고려해 제출 시점 로그인으로)
- [ ] **필수항목 최소화** — 현재 필수 6개(제목·카테고리·장르·등급·해상도·AI도구). 해상도·AI도구는 자동감지 → "자동 감지됨" 강조, 카테고리·장르 추천 기본값. **"나머지는 나중에 편집" 허용.**
- [ ] **3분 미만 라이선스 불가 사전 안내** — Step 1 파일 선택 시 길이 경고(현재 Step 3에서야 노출, `Upload.tsx:1840`).
- [ ] **업로드 직후 = AI 홍보문건 원클릭** — 기존 `/generate-promo`(VideoEditModal) 연결 → 캐치프레이즈·SNS캡션·해시태그 자동 생성 → **"내 SNS에 바로 공유"** 버튼. (크리에이터가 자기 영상 홍보 = 우리 시청자 유입까지 동반)
- [ ] **초기 크리에이터 배지/혜택 배너** — 업로드 화면 상단에 "지금 입점 = 초기 크리에이터 혜택" 고지.
- [ ] **정산 안내 명확화** — "업로드는 무료·무조건 가능, 정산(현금화)은 ₩10k 이상+세금정보 등록 시, 실지급은 결제 오픈 후" 솔직 고지.

*콜드스타트는 자동화로 안 풀린다. 첫 30~50명은 손으로 모은다.*

  - 패스트캠퍼스·인프런 등 **AI 영상 강의 수강생** 커뮤니티.
- [ ] **1:1 온보딩** — 우리가 업로드 직접 도와주고 첫 콘텐츠 품질 관리(초기 인상 = 전부).

## 3. 출시 순서 연동 (병목 고려)

- **무료 광고형 티어는 토스 무관 → 크리에이터 유치도 선출시 가능.** 지금부터 Phase 0~1 진행 가능.
- **실결제·현금 정산(특히 라이선스 판매)은 토스 가맹 후행** → 초기엔 "올려두면 결제 오픈 시 바로 판매·정산" 으로 **정직하게** 안내(과약속 금지).
- 시청자 측 홍보 자산(전단 4종 `marketing/`)과 양면 동시 추진.
