## 출원 절차 (특허로)
1. 특허고객번호 발급(최초 1회) + 공동인증서 로그인
2. 상표 → 국내출원 → 출원서 작성 (상표 견본 업로드, 위 지정상품 선택, 출원인=크레비즈)
3. 수수료 납부: 출원료 류당 약 ₩56,000~62,000(전자), 등록결정 후 등록료 류당 약 ₩21만(10년)
4. 심사 8~14개월. 의견제출통지(거절이유) 시 답변서 제출.

## 비용 개략 (셀프, 전자출원)
- 1개 상표 × 1개 류 ≈ 출원 6만 + 등록 21만 = **약 27만원**
- 권장 구성: (영문 + 한글) × (41 + 42류) = 4건 → 대략 출원 24만 + 등록 84만 = **약 108만원**
  (광고수익 본격화 시 35류, 앱 출시 시 9류 추가)
